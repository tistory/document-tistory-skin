소개

티스토리 스킨 가이드

티스토리에서는 스킨을 통해서 사이트를 자유롭게 디자인할 수 있습니다. 스킨은 치환자를 사용하여 제작되고 사용된 치환자에는 티스토리의 데이터가 대입되어 티스토리의 디자인이 완성됩니다.

이 문서는 스킨에서 사용하는 치환자에 대해 설명합니다.
